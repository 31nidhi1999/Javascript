Implemented a simple counter using JavaScript that updates dynamically based on user interaction. Three buttons (increase, decrease, and reset) allow users to modify the counter value. The text color changes based on the value:

Green for positive numbers

![increment](https://github.com/user-attachments/assets/029e0cd1-a980-4223-8236-9eacfdb2e3d4)


Red for negative numbers

![decrement](https://github.com/user-attachments/assets/8589eb98-b960-42e3-9e62-530cc51ead28)


Black when the count is zero

![image](https://github.com/user-attachments/assets/9d5e705d-5301-4a0e-8b99-20474e17e72e)


This feature uses querySelector and addEventListener for DOM manipulation and event handling.
